# vim Lesson5

## 5-1. 외부 명령 실행하는 방법

- :!command 를 이용하여 외부 명령을 실행한다.


## 5-2. 보다 자세한 파일 저장

- :w FILENAME 하면 현재 vim에서 사용하는 파일을 FILENAME이라는 이름으로 저장한다.


## 5-3. 선택적으로 저장하는 명령

1. v(visual)를 누르고 저장할 공간만큼 하이라이트를 시킨다.
2. :를 누르면 화면 하단에 :'<'>가 나타난다.
3. '<'>w FILENAME 입력되었으면 FILENAME으로 저장된다.

## 5-4. 파일 읽어들이기, 합치기

- :r FILENAME 은 디크스에서 FILENAME이라는 파일을 불러들여서 커서 위치 뒤에 현재 파일을 집어넣는다.
